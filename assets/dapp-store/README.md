# dApp Store Assets

Place your submission assets here. See `docs/dapp-store-assets.md` for dimensions and requirements.

## Structure

- `icon/` — 512×512 PNG (copy from `public/icons/icon-512.png`)
- `banner/` — 1200×600 PNG/JPG
- `feature/` — 1200×1200 PNG/JPG (optional, for Editor's Choice)
- `screenshots/` — 1920×1080 PNG/JPG (min 4, all same orientation)
- `video/` — optional demo video
